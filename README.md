## Run Locally

Clone the project

```bash
https://github.com/vinayakpagar23/Travelopia.git
```

Go to the project directory

```bash
  cd travelopia
```

Install dependencies

```bash
  npm install
```

```bash
  cd frontend/
  npm install
```

Start the server

```bash
  npm run start
```
Start the frontend

```bash
  //open now terminal
  cd frontend
  npm start
```
